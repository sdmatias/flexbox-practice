# Proyecto Flexbox - Layout Responsivo
Esteproyecto esta enfocado en la práctica de Flexbox, diseño mobile first y layout adaptativo. Se incluyen componentes como navbar fijo, galería, y sidebar responsiva.


## Tecnologías


- HTML5
- CSS3 (Flexbox, Media Queries)
- Animaciones con AOS.js

##Objetivos

- Dominar `flex-direction`, `justify-content`, `align-items` y `flex-wrap`
- Implementar estructuras semánticas y escalables
- Aplicar efectos visuales modernos y accesibilidad
